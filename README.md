# Modern Seaside Stay

A **modern, responsive, and interactive seaside accommodation website** built with React, Vite, Tailwind CSS, and shadcn-ui.  
This project showcases a fully responsive UI for a seaside stay, including gallery sections, location maps, and interactive components.

## Project Overview

Modern Seaside Stay is designed to provide an immersive, user-friendly experience for potential guests.  
The project focuses on **clean UI, responsive design, and modern web technologies**.

Key objectives:

- Showcase amenities through a dynamic gallery.  
- Display property location using an interactive Google Map embed.  
- Provide a seamless experience across desktop, tablet, and mobile devices.

---

## Features

- Fully responsive layout using **Tailwind CSS**.  
- **Gallery section** with hover effects and optimized images.  
- **Location section** with Google Maps embed.  
- Smooth **animations and transitions**.  
- Built with **React** and **Vite** for fast development and build times.  
- Accessible and optimized for performance.  

---

---

## Technologies Used

- **React** - Frontend library for building UI components  
- **Vite** - Fast development and build tool  
- **TypeScript** - Static type checking  
- **Tailwind CSS** - Utility-first CSS framework  
- **shadcn-ui** - Prebuilt accessible UI components  
- **Google Maps Embed** - Location visualization
